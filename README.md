# CalculoNumerioAplicado
Repositório dedicado à todas as atividades, projetos e códigos feitos no curso de cálculo numérico aplicado.

A APC1 tem como objetivo mostrar que a equação dimensional do movimento da partícula pode ser adimensionalizada em funcção do número de Stokes e o número de Reynolds, para mais informações relacionadas à esse projeto, aguarde as próximas atualizações. Keep chill!

O PPC1 é um programa em python que procura mostrar o que foi apresentado na APC1, junto do PPC1 está o PPC1.pdf que analisa e discute o resultado da execução do código e gráfico obtido.
